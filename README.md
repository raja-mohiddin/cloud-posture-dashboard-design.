### Project Overview
Security teams lack centralized visibility across multi-cloud environments (AWS, Azure, GCP). This dashboard provides a unified view that normalizes misconfiguration data, allowing security architects to instantly filter by severity and cloud provider to prioritize remediation.

### Key Features
1. **Unified Filter Sidebar:** Allows rapid sorting across thousands of resources.
2. **Actionable Remediation Drawer:** Provides direct CLI code snippets to fix misconfigurations, reducing the friction between Security and DevOps.
3. **Global Posture Score:** High-level widgets for immediate executive visibility.
